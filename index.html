Sinkronisasi Cloud Google Drive

Cadangkan seluruh data ke Google Drive pribadi Anda untuk sinkronisasi antar-perangkat secara gratis dan aman.

Client ID tersimpan lokal di browser ini.
Cara Membuat Client ID secara Gratis:
  1. Buka Google Cloud Console.
  2. Buat Project Baru di bagian header atas.
  3. Masuk ke menu APIs & Services ➔ OAuth Consent Screen. Pilih tipe User External, isi nama aplikasi ("YosDay") dan email Anda. Simpan & lanjutkan.
  4. Lalu masuk ke APIs & Services ➔ Credentials. Klik + Create Credentials ➔ OAuth Client ID.
  5. Pilih Application Type: Web Application.
  6. Di bagian Authorized JavaScript origins, tambahkan URL asal web Anda. Contoh:
    • http://localhost:8080 (jika via local server default)
    • http://127.0.0.1:5500 (jika via VS Code Live Server)
    • https://yosday-anda.vercel.app (jika sudah di-deploy)
    Penting: Google melarang protokol file:///, jadi web wajib dijalankan lewat server.
  7. Klik Create, lalu salin Client ID yang berakhiran .apps.googleusercontent.com.
